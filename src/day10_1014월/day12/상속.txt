- 상속이란?
    - 상위클래스가 하위 클래스의 개념을 확장, 세분화
    - (현실) 부모로부터 자식 에게 물려주는 행위
    - (자바) 부모클래스가 자식 클래스에게 멤버변수/멤버함수 물려주는 행위
    - 클래스 이란? 객체를 실체화(물리적) 하기 위한 객체 설계도
        - 객체 이란? 속성과 기능을 가진을 물리적 존재 이거나 추상적인 개념
    - (클래스 상속) 클래스의 속성(멤버변수) 정의 와 기능(메소드) 를 다른 클래스에게 확장하는 개념

    - 생활속에서 상속을 물리적으로 표현하는 방법
        - 부모님의 통장(속성)을 자식이 물려받는다. - 자식은 물려받은 부모의 통장(속성)을 사용할수 있다.

- 자바에서 상속 표현하는 방법
    - (현실) 부모가 자식을 낳아서 선택하지만..
    - (자바) *자식이 부모를 선택한다. *자식이 부모를 선택해서 연장한다.
    [1] extends : 연장/확장 하다.
        class 부모클래스 { }
        class 자식클래스 extends 부모클래스{ }
    [2] 논리적인 개념
        - 자식 클래스는 부모 클래스의 멤버변수 와 메소드를 사용할수 있다.
            - 부모 클래스는 자식 클래스의 멤버변수 와 메소드를 사용할수 없다.
            - 자바는 자식이 부모를 선택하기 때문에. 자식은 부모를 알지만 부모는 자식을 모른다.
        - 자식 클래스는 부모 객체의 멤버변수와 메소드를 사용하지만, 형제는 불가능하다.
    [3] 물리적인 개념
        * 기차 : 서로 다른 열차 칸 들을 하나로 연결된 구조
        - 자식 클래스는 자식 객체를 생성할때 부모 클래스의 부모 객체도 함께 생성한다.
            - 부모 클래스는 부모 객체를 생성할때 자식 클래스의 자식 객체도 함께 생성 하지 않는다.
        - 자식 클래스는 자식 객체를 생성할때 부모객체를 먼저 생성하고 자식 객체를 생성한다.
        예시] 동물 클래스를 객체 생성
            'Object' 클래스 객체 <---- '동물' 클래스 객체
        예시] 조류 클래스를 객체 생성
            'Object' 클래스 객체 <---- '동물' 클래스 객체 <---- '조류' 클래스 객체
        예시] 어류 클래스를 객체 생성
            'Object' 클래스 객체 <---- '동물' 클래스 객체 <---- '어류' 클래스 객체

    - Object 클래스란?
        - 자바에 미리 만들어진 클래스로 최상위 클래스 입니다.
        - 자바의 모든 클래스들은 extends 없어라도. Object 클래스로 부터 상속을 받는 구조 입니다.
        - 왜?? 다형성를 표현 할수 있다.
    - 다형성 이란?
        - 다양한 형태의 성질
        - 자료의 타입 변환 하는 하는 행위.
