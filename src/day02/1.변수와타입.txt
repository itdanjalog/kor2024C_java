[1] 변수
    - 변수(변하는 수) vs 상수(고정된 수)
    - 상징적인이름 = 자료
        - 예시] 도토리필드 = 6(자료)
    - 하나의 자료를 저장 할수 있는 메모리(컴퓨터저장기능) 공간
        - 변수명은 메모리 공간의 상징적인 이름
        - 메모리 공간(변수) 안 에는 자료/리터럴 가 1개 저장된다.
    - 변수 선언하는 방법
        1. 타입을 선택한다. ( * 고려할점 : 자료의 크기 )
               ex] '시계'선물하기 위해 다이소에 가서 선물상자를 고른다. ( 적절한 시계를 포장할 상자 )
        2. 변수명을 정의한다. ( 변수의 상징적인 이름 , 식별/구분 역할 )
                ex]    엄마시계상자=( '시계' )    아빠시계상자=('시계')
            - 변수명의 주의할점 # 암기x # 코드 하면서 문제 발생시 대처
                1.중복이 안된다.
                2.특수문자(띄어쓰기포함) 제외 # $ _ 가능하다.
                3.숫자로 시작할수 없다.
                4.관례적으로 카멜표기법 권장( 클래스명,함수명,변수명,상수명 구분 하기 위해서 )
                    카멜표기법 : sportscar -> sportsCar
                    * 첫번째 단어는 소문자로 시작 하고 두번째 단어의 첫글자를 대문자
        3. 변수에 자료를 대입한다.
            =  : 오른쪽 자료를 왼쪽에 대입/저장 한다.
        4. 변수에 대입할 자료/리터럴 값
            변수에 저장할 자료 / 값
[2] 타입
    - 자료 들의 분류
    - 효율적인 자료를 다루기 위해서 사용한다. # 가독성(읽기) 뛰어나다. # 데이터의 적절한 타입 분류
    - 종류 : 기본타입 vs 참조타입(추후 학습)
    - 기본타입 [8가지] 종류
                        키워드             용량                자료 범위
        1. 불리언타입   boolean             1바이트             true , false
        2. 정수타입
                        byte              1바이트             -128 ~ 127  ( 대략 +-100 )
                        short             2바이트             -32768 ~ 32767 ( 대략 +-3만정도 )
                        int               4바이트             대략 +-21억정도 , *리터럴의 기본 정수 타입
                        long              8바이트             대략 +-21억이상
        3. 실수타입
                        float             4바이트              소수점 8 자리 표현
                        double            8바이트              소수점 17 자리 표현

        4. 문자타입      char               2바이트              ''작따옴표 , 영문/특수문자 은 1바이트 , 한글 2바이트
        + 문자열타입     String             클래스               " "큰따옴표 ,  "유재석" * 2 -> 6바이트


- true 와 false 는 0 과 1 이라고 하는데 왜 1비트( 0 또는 1 ) 가 아니고 1바이트 생각
