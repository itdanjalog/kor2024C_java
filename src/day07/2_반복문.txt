제어문 종류 : 1. 조건문 2.반복문
[반복문]
    - 반복(되풀이) 한다.
    - 패턴 : 어디서부터 어디까지 증감하면서 반복하는지 패턴 찾기
        반복변수(i) : 어디서부터 어디까지 증(가)감(소) 하는 주체자(누가)
            - 유재석이 1개의계단부터 10개의계단까지 계산1칸씩 오른다를 반복한다.
            for( int 유재석 = 1 ; 유재석 <= 10 ; 유재석++ ) {
                오른다;
            }

    - 문법 :
        for( 초기값 ; 조건식; 증감식 ){
            반복실행문;
        }
