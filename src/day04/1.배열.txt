[0] 변수
    - 변수 : *하나의 자료* 를 저장하는 메모리 공간
    - 변수명 : 그 메모리 공간의 상징적인 이름

[1] 배열
    - 여러개의 자료들을 *순서대로* 저장하는 자료 타입(참조 타입)
    - *배열타입 또한 하나의 자료이다.
    - 주의할점 :
        1. 여러개의 자료들은 모두 동일한 타입이여아한다.
        2. 배열이 만들어질때 정해진 자료 개수는 수정이 불가능하다.(고정길이)
    - 배열(참조)타입 # 타입 옆에 [ ] 붙인다.
        - int[] : 정수 자료 여러개를 저장할수 있는 타입
        - double[] : 실수 자료 여러개를 저장할수 있는 타입
            - boolean[] , byte[] , short[] , long[] , float[]
        - char[] : 문자 자료 여러개를 저장할수 있는 타입 *************
            - 문자 여러개는 문자열 # String 타입의 기반이 되는 문자 배열
    - 배열 자료를 정의 하는 방법
        방법1. new 타입[ 자료개수 ];        # 배열의 길이/개수만 정의하고 배열내 내용물 없는 상태
        방법2. { 자료1 , 자료2 , 자료3 }    # 배열를 정의할때 배열내 내용물를 같이 정의한 상태
    - 배열 자료를 변수에 저장하는 방법
        타입[ ] 변수명 = new 타입[ 자료개수 ]
        타입[ ] 변수명 = { 자료1 , 자료2 , 자료3 }
    - 인덱스 : 자료가 저장된 위치 번호 # 배열은 저장순서 대로 부여한다
        - 자료 하나씩 순서대로 0번부터 부여한다.
        - 배열내 여러개 자료들을 식별하기 위해서 사용한다.
            - 배열내 자료들이 중복 자료 를 허용하기 때문에 중복자료를을 개별 식별
    - 배열내 요소(자료) 를 호출/꺼내는/반환 하는 방법
        - 배열변수명[인덱스]
            1. 요소 값 수정/대입 : 배열변수명[인덱스] = 새로운값
            2. 요소 값 호출 : 배열변수명[인덱스]
            3. 요소 값 삭제 : 없음 # 배열 요소를 삭제할수 없다 #배열은 고정길이
