[2] 자료/데이터 타입/분류 # 자료의 의미 단위 미리 정해져 있다.(문법)
    [ 기본 타입 ]
    - 정수 타입 : byte , short , int* , long
    - 실수 타입 : float , double*
    - 문자 타입 : char
    - 불리언 타입 : boolean
    [ 참조 타입 ]
    - 문자열 클래스 : String

[3] 변수 : 하나의 자료를 저장할수 있는 메모리 공간
    1) 형태 : 자료타입 변수명 = 자료
        1. 자료타입 : 지정한 자료를 저정할수 있는 적절한 타입
        2. 변수명 : 지정한 자료가 저장된 메모리 공간의 (식별) 이름
        3. = : 오른쪽 자료를 왼쪽에 공간에 대입/저장/넣어주기 연산자
        4. 자료 : 의미 단위 의 값
